# Goal: TC 结构化映射关系治理

## 角色定义
**身份**：AI需求全流程治理工程师（资深测试工程师 + 高级prompt设计师）

## 原始目标
建立测试用例内部结构化映射关系，解决 Excel 生成质量差、结构混乱的问题：
- 1个用例描述 → N个功能描述
- 1个功能描述 → N个前置条件
- 1个前置条件 → N个操作步骤
- 1个预期 → N个操作步骤

## 验收标准 (accept_criteria)

| # | 标准 | 验证方式 |
|---|------|----------|
| 1 | 产出 TC 内部结构化映射规范文档 | 文件存在 + 内容完整 |
| 2 | 修改 S6 SKILL.md 集成映射规则 | grep 检查关键字段 |
| 3 | 修改 test_case_formatter.py 支持新映射 | 代码含映射逻辑 |
| 4 | 修改 test_cases.json schema | schema 含层级字段 |
| 5 | 重新生成 v3.01 测试用例验证 | TC 数量/结构符合预期 |
