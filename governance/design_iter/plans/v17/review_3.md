## 4. 立即可执行的优化

### 优化1: 添加 L1 校验（§12 规范）

在 `ai_workflow/validators/l1_s6.py` 中添加：

```python
def validate_structural_mapping(test_cases):
    """校验 §12 TC 结构化映射规范"""
    errors = []
    for tc in test_cases:
        steps = tc.get('操作步骤', [])
        expected = tc.get('预期结果', [])
        
        # 检查步骤数
        if len(steps) < 3:
            errors.append(f"{tc['case_id']}: 步骤数 {len(steps)} < 3")
        
        # 检查预期结果格式
        for exp in expected:
            if isinstance(exp, dict) and 'step_ref' not in exp:
                errors.append(f"{tc['case_id']}: 预期结果缺少 step_ref")
    
    return errors
```
