### OBJ-01: 商城首页
- **所属角色**: ROLE-01/04/05
- **模块**: UI
- **功能列表**:
  1. 热门道具推荐展示（前 10 按销量排序）
  2. 分类导航栏（武器/时装/坐骑/消耗品/礼包）
  3. 道具名称模糊搜索
  4. 分页列表（每页 20 个）
  5. 推荐算法排序
- **业务故事**:
  - 故事 1: 作为普通玩家，我希望浏览热门推荐，以便快速发现热卖道具
  - 故事 2: 作为普通玩家，我希望按分类导航浏览，以便快速找到目标类型
  - 故事 3: 作为普通玩家，我希望通过关键词搜索道具，以便快速定位目标
  - 故事 4: 作为普通玩家，我希望按页翻看道具列表，以便浏览更多道具
  - 故事 5: 作为灰度账号，我希望推荐按销量排序验证，以便确认排序规则
- **数据流转 / 落地**:
  - 入口: 推荐服务 / 销量统计表
  - 处理: 排序算法 / 搜索服务
  - 出口: 渲染 UI 列表
- **风险点**: 10000 并发下推荐服务响应时间、搜索性能
