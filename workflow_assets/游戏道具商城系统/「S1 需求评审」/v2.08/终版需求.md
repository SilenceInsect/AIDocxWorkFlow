# 游戏道具商城系统 — 终版需求 v1.0

> S1.5 业务澄清后定稿（重跑于 2026-06-15）

## 1. 用户角色

- **普通玩家**：浏览商城、购买道具、查看订单
- **VIP 玩家**：享专属折扣（VIP1:95折 / VIP2:9折 / VIP3:85折）+ 限时道具
- **管理员**：上下架道具、配置价格、管理促销

## 2. 功能模块

### F1: 商城首页
- 展示热门道具推荐（前 10 个按销量排序）
- 分类导航：武器 / 时装 / 坐骑 / 消耗品 / 礼包
- 搜索功能：名称模糊搜索（名称 70% + 描述 30% 权重）
- 每页展示 20 个道具：名称 / 图标 / 价格 / 折扣标签

### F2: 道具详情页
- 道具名称、描述、属性加成、价格
- 角色穿戴预览
- 购买数量 1-99
- 余额不足时按钮置灰 + Toast 提示

### F3: 购买流程
- 支付前确认：道具名称 / 数量 / 总价
- 支付方式：游戏币（默认）/ 人民币
- 道具到账：0 延迟
- 邮件通知：购买成功后立即发送

### F4: 订单管理
- 查询最近 30 天订单
- 订单字段：订单号 / 道具名 / 数量 / 金额 / 支付方式 / 时间
- 筛选：时间范围 + 道具名称

### F5: VIP 专属
- 折扣梯度：VIP1 95折 / VIP2 9折 / VIP3 85折
- VIP 专属道具仅 VIP 可见
- VIP 折扣不与限时打折叠加（**取较低价**）

### F6: 促销
- 限时打折：指定时间窗内生效
- 满减梯度：100-10 / 200-25 / 500-60
- 新手礼包：新注册 7 天内可购买 1 次

## 3. 非功能需求

- 页面加载 < 2 秒
- 接口响应 < 500ms
- 10000 并发用户
- 缓存 5 分钟（**击穿用互斥锁单飞**）
- 所有支付操作记录日志
- 退款策略：24h 内未使用道具可全额退款
- 灰度策略：5% → 20% → 100% 三阶段
- 限流：单 IP 100 QPS 软限 + 全局 10000 并发硬限

## 4. 验收标准

- ✅ 商城首页可浏览所有分类
- ✅ 游戏币购买成功到账 < 3 秒
- ✅ VIP 自动享受对应折扣
- ✅ 限时促销期间显示折扣标签
- ✅ 订单可查 + 可筛选
- ✅ 余额不足时按钮置灰 + 提示
