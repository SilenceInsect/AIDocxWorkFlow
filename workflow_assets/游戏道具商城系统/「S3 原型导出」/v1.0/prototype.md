## PAGE-001：商城首页

- **URL**：`/shop`
- **入口**：玩家登录后自动进入
- **核心功能**：热门道具展示、分类导航、搜索入口、道具列表

### 布局结构

```
+------------------------------------------------------+
| Logo | [搜索道具...]                    | 用户头像      |
+------------------------------------------------------+
| [武器] [时装] [坐骑] [消耗品] [礼包]  <- 分类导航栏  |
+------------------------------------------------------+
| 🔥 热门道具                                           |
| [卡片1][卡片2][卡片3][卡片4][卡片5]  <- 横向滚动    |
+------------------------------------------------------+
| 分类道具列表（分页 20个/页）                         |
| [道具卡片][道具卡片][折扣卡片][道具卡片]              |
|                                                 ... |
+------------------------------------------------------+
|                    < 1  2  3  ... 10 >              |
+------------------------------------------------------+
```

### 交互元素

| 元素 | 行为 |
|------|------|
| 搜索框 | 最小2字符触发，实时下拉搜索结果 |
| 分类标签 | 点击切换，URL参数变化，列表刷新 |
| 热门道具卡片 | 点击跳转详情页，有hot角标 |
| 道具卡片 | 点击跳转详情页，显示已折后价 |
| 折扣角标 | 红色背景，显示折扣比例（如85折） |
| 分页控件 | 切换页面，URL参数变化 |

### 状态说明

- **空数据**：分类下无道具，显示「暂无道具」
- **加载中**：骨架屏占位，不显示真实内容
- **搜索无结果**：显示「未找到匹配的道具」

---

## PAGE-002：道具详情页

- **URL**：`/shop/item/{item_id}`
- **入口**：商城首页道具卡片点击
- **核心功能**：道具信息展示、数量选择、VIP折扣展示、购买入口

### 布局结构

```
+------------------------------------------------------+
| <- 返回商城                             | 用户头像      |
+------------------------------------------------------+
| +-----------------+ +------------------------------+ |
| |                 | |  道具名称                       | |
| |   道具图标       | |  描述：属性加成说明...           | |
| |   (点击放大)     | |                               | |
| +-----------------+ | +---------------------------+  | |
|                   | | | 折扣标签: 限时85折          |  | |
|                   | | | VIP标签: VIP3 享85折        |  | |
|                   | | | 原价¥120 -> 折后¥88        |  | |
|                   | | +---------------------------+  | |
|                   | |                               | |
|                   | | 数量: [ - ] [ 1 ] [ + ]       | |
|                   | | 小计: ¥88 × 1 = ¥88          | |
|                   | |                               | |
|                   | | 💰 余额: ¥500  ✓ 充足          | |
|                   | | [ 立即购买 ]                    | |
|                   | |                               | |
|                   | +------------------------------+ |
+------------------------------------------------------+
```

### 交互元素

| 元素 | 行为 |
|------|------|
| 道具图标 | 点击放大预览Modal |
| 数量步进器 | 点击-/+调整1-99，超出前端拦截+Toast提示 |
| 总价实时计算 | ¥单价×数量，余额不足时购买按钮禁用 |
| VIP专属道具 | 非VIP用户访问显示权限不足提示 |
| 购买按钮 | 余额充足蓝色可点，不足灰色禁用+提示 |
| 返回按钮 | 返回上一页 |

### 状态说明

- **非VIP用户**：不显示VIP专属道具；若已到详情页则提示权限不足
- **余额不足**：购买按钮禁用，显示「余额不足，请先充值」
- **道具售罄**：购买按钮禁用，显示「该道具已售罄」
- **有VIP折扣**：金色VIP角标+折后价+VIP专属折扣说明

## PAGE-003：购买确认弹窗（Modal）

- **入口**：道具详情页购买按钮点击
- **核心功能**：确认购买信息、余额对比、支付方式选择、折扣叠加展示

### 布局结构

```
+------------------------------------------+
|  确认购买                              [×] |
+------------------------------------------+
|  道具名称：屠龙刀 × 1                   |
|  -----------------------------------     |
|  原价：¥120                             |
|  促销折扣：限时85折 -> -¥18             |
|  VIP折扣：VIP3 85折 -> -¥12            |
|  -----------------------------------     |
|  应付金额：¥90  <- 取最优价 min(折后1,折后2) |
|  -----------------------------------     |
|  💰 账户余额：¥500                       |
|  💰 购买后余额：¥410                     |
|  -----------------------------------     |
|  支付方式：                             |
|  (●) 游戏币支付                         |
|  ( ) 人民币支付                         |
|  -----------------------------------     |
|  [取消]                 [确认购买 ¥90]   |
+------------------------------------------+
```

### 交互元素

| 元素 | 行为 |
|------|------|
| 支付方式切换 | 点击单选切换，余额显示实时更新 |
| 确认购买 | 余额不足时按钮禁用 |
| 关闭按钮(×) | 关闭弹窗，返回PAGE-002 |

### 状态说明

- **余额不足**：应付金额红色高亮，确认购买按钮禁用
- **有促销**：折扣明细单独一行
- **有VIP折扣**：VIP折扣单独一行，与促销折扣分别列出

## PAGE-005：支付失败页

- **URL**：`/shop/pay/fail?order_id={order_id}&reason={reason}`
- **入口**：支付失败回调
- **核心功能**：失败原因展示、重新支付入口

### 布局结构

```
+------------------------------------------------------+
|                                                      |
|              ❌ 支付失败                             |
|                                                      |
|  失败原因：余额不足                                  |
|  （reason参数展示具体错误信息）                      |
|                                                      |
|       [重新支付]           [返回商城]               |
|                                                      |
+------------------------------------------------------+
```

### 状态说明

- 失败原因根据URL `reason`参数展示（余额不足/支付超时/渠道异常等）

## Mermaid 页面流图

```mermaid
flowchart TD
    Start([玩家进入商城]) --> Home[商城首页<br/>PAGE-001]
    Home --> Search[/shop?search=keyword]
    Home --> Category[/shop?category=xxx]
    Home --> Hot[热门道具卡片]
    Hot --> Detail[道具详情页<br/>PAGE-002]
    Category --> Detail
    Search --> Detail
    Detail --> Confirm[购买确认弹窗<br/>PAGE-003]
    Confirm --> PayGame[游戏币支付]
    Confirm --> PayRMB[人民币支付<br/>渠道选择]
    PayGame --> Success[支付成功页<br/>PAGE-004]
    PayRMB --> Success
    PayGame --> Fail[支付失败页<br/>PAGE-005]
    PayRMB --> Fail
    Fail --> Confirm
    Success --> Orders[订单列表页<br/>PAGE-006]
    Success --> Home2[返回商城<br/>PAGE-001]
    Orders --> OrderDetail[订单详情页<br/>PAGE-007]
    OrderDetail --> Orders

    Hot -.-> |VIP专属| NoAccess[权限不足提示]
    Confirm -.-> |余额不足| Confirm
    Detail -.-> |余额不足| Alert["余额不足<br/>购买按钮禁用"]
```
