# H. 边界异常 UI 场景

> **子类代码**：`EDGE_UI`（聚合 3 子项：极端数据 / 网络异常 / 权限控制）
> **归属模块**：`UI`
> **来源**：`.cursor/MODULES.md` §4.5 H
>
> **测什么**：极端数据（超长/空/海量）、网络异常（断网/超时）、权限控制（无权限/未解锁）的**UI 占位**。
> **不测什么**：业务异常逻辑（归 SPECIAL）、网络逻辑（归 UTIL）、权限校验（归 BIZ）。
> **与其他子类的差异**：H 测"异常状态的 UI 占位"——D 测正常状态的资源展示；H 测视觉、UTIL 测逻辑。

---

## 1. 典型场景

### 场景 1：超长名称
- 业务背景：玩家排行榜
- 涉及元素：玩家名
- 验证点：50 字玩家名不溢出容器、省略号处理

### 场景 2：超大数字
- 业务背景：货币显示
- 涉及元素：货币数值
- 验证点：99999999 显示完整、超过 9 位用 k/m/w 简写（按设计）

### 场景 3：空列表
- 业务背景：商品列表无数据
- 涉及元素：列表、空状态占位
- 验证点：显示"暂无商品"、推荐按钮

### 场景 4：上万条列表
- 业务背景：商品列表
- 涉及元素：虚拟滚动
- 验证点：渲染流畅（不卡顿）、滚动帧率 ≥ 30fps

### 场景 5：断网提示
- 业务背景：网络断开
- 涉及元素：网络异常 UI
- 验证点：断网提示横幅、离线模式入口

### 场景 6：加载失败重试
- 业务背景：接口超时
- 涉及元素：失败占位
- 验证点：失败占位 UI、点击重试按钮

### 场景 7：超时占位
- 业务背景：加载超时
- 涉及元素：超时占位
- 验证点：超时后切换到占位 UI

### 场景 8：无权限按钮
- 业务背景：未登录访问付费功能
- 涉及元素：按钮
- 验证点：按钮置灰、显示"请登录"提示

### 场景 9：隐藏付费入口
- 业务背景：未成年用户
- 涉及元素：付费按钮
- 验证点：付费入口完全隐藏（不渲染）

### 场景 10：未解锁控件
- 业务背景：等级不足
- 涉及元素：功能按钮
- 验证点：按钮置灰 + 锁图标、hover 提示"等级不足"

## 3. 边界陷阱

### 边界 1：vs D. 静态展示
- **混淆点**：「空列表"占位"」——D 测空数据样式、H 测边界
- **判定规则**：测"空数据时的 UI 占位样式" → D；测"加载失败/超时等异常状态" → H
- **实例**：空数据占位图 → D-006；加载失败占位 → H-007

### 边界 2：vs UTIL
- **混淆点**：「网络断开"」——UTIL 测检测逻辑、H 测 UI
- **判定规则**：测"网络断开检测/重试机制" → UTIL；测"断网提示横幅 UI" → H
- **实例**：断网检测回调 → UTIL；断网横幅样式 → H-006

### 边界 3：vs BIZ
- **混淆点**：「无权限"按钮"」——BIZ 测权限校验、H 测 UI 占位
- **判定规则**：测"权限校验逻辑（前后端）" → BIZ；测"无权限时按钮 UI 状态" → H
- **实例**：后端 401 响应 → BIZ；按钮置灰 + 提示 → H-009

### 边界 4：vs SPECIAL
- **混淆点**：「断网"处理"」——SPECIAL 测网络异常处理、H 测 UI
- **判定规则**：测"断网时业务降级逻辑" → SPECIAL；测"断网占位 UI" → H
- **实例**：断网时使用本地缓存 → SPECIAL；显示断网横幅 → H-006
