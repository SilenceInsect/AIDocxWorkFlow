# O. UI 模块边界区分（判定规则）

> **非测试类型**——本文件是 S5 生成测试点时的**判定参考**，
> 避免误标 UI 标签（实际是 CONFIG / BIZ / HINT / AUX / LINK / LOG / SPECIAL）。

---

## 1. 判定原则

> **UI = 页面视觉 + 前端交互 + 布局适配 + 静态展示 + 动效 + 无障碍 + 异常 UI 场景**
>
> **不归 UI**：后端业务逻辑、配置内容、提示逻辑、底层工具

### 判定三问

1. **是"纯前端 UI 层"还是"后端业务逻辑"？** 是后端 → 不归 UI
2. **是"UI 展示/样式/交互"还是"内容/文案/触发条件"？** 是内容 → 不归 UI
3. **是"页面渲染"还是"底层资源加载"？** 是底层 → 不归 UI

## 2. 边界对照表

### vs BIZ

| 场景 | 归 UI | 归 BIZ |
|------|-------|--------|
| 购买按钮 | 按钮样式、点击态 | 购买业务逻辑、扣款发货 |
| 商城列表 | 商品卡片渲染、列表滑动 | 商品数据加载、下单接口 |
| 任务列表 | 任务卡片 UI、进度条动画 | 任务进度更新、奖励发放接口 |
| 战斗 UI | 血条、角色模型、技能图标 | 伤害计算、暴击逻辑 |
| 邮件列表 | 邮件列表 UI、附件按钮样式 | 邮件收发、附件领取接口 |
| 道具背包 | 道具图标、Tooltip 样式 | 道具产出消耗逻辑 |
| 聊天 | 聊天界面 UI、输入框样式 | 聊天消息收发、协议 |
| 排行榜 | 排行榜 UI、排名展示 | 排行榜数据获取、赛季结算 |

### vs HINT

| 场景 | 归 UI | 归 HINT |
|------|-------|---------|
| 红点 | 红点图标样式、颜色、位置 | 红点显示条件、触发逻辑、内容文案 |
| 飘字 | 飘字动画效果、位置、时长 | 飘字内容（+100 金币）|
| Toast | Toast UI 样式、出现位置 | Toast 文案内容、触发条件 |
| 弹窗 | 弹窗布局、按钮样式、遮罩颜色 | 弹窗文案内容、触发条件 |
| 引导 | 引导遮罩颜色/透明度、高亮区域样式 | 引导文案内容、引导步骤顺序 |
| 角标 | 角标图标样式、颜色 | 角标数字内容 |

**核心区分**：**临时弹出 = HINT；常驻展示 = UI**

### vs CONFIG

| 场景 | 归 UI | 归 CONFIG |
|------|-------|-----------|
| 图标 | 图标渲染、显示效果 | 图标路径字段、路径存在性 |
| 皮肤 | 皮肤切换动效、UI 展示 | 皮肤表字段、皮肤 ID 合法性 |
| 文案 | 文案在 UI 上的展示效果 | 文案 key 字段、国际化映射 |
| 颜色 | 颜色在 UI 上的应用效果 | 颜色值字段（#RRGGBB）|
| 字体 | 字体渲染效果 | 字体名称/路径字段 |

### vs AUX

| 场景 | 归 UI | 归 AUX |
|------|-------|--------|
| 资源加载 | 资源加载后的展示效果 | 资源下载、CDN 路径、缓存命中率 |
| 弹窗封装 | 弹窗 UI 组件样式 | 弹窗 API/框架 |
| 加载动画 | 加载动画 UI | 加载工具/骨架屏框架 |
| 路由 | 页面跳转效果 | 跳转路由 API |
| 性能 | 性能 UI 指标显示 | FPS/内存监控组件 |

### vs LINK

| 场景 | 归 UI | 归 LINK |
|------|-------|--------|
| 多端展示 | 多端 UI 一致性展示 | 多端数据同步逻辑 |
| 跨服展示 | 跨服数据 UI 展示 | 跨服数据同步时序 |

### vs LOG

| 场景 | 归 UI | 归 LOG |
|------|-------|--------|
| 操作日志 | UI 上展示的日志记录 | 日志埋点内容、上报逻辑 |
| 崩溃展示 | UI 崩溃提示样式 | 崩溃堆栈上报内容 |

## 3. 常见误判案例

### 误判 1：红点显示条件 → 错标 UI

- **错误**：看到"红点"，就标 UI
- **正确**：
  - UI 测"红点图标样式、颜色、位置"（归 UI）
  - HINT 测"红点显示条件、触发逻辑、内容文案"（归 HINT）
- **区分**：测"样式" → UI；测"为什么显示/显示什么" → HINT

### 误判 2：飘字内容 → 错标 UI

- **错误**：看到"飘字"，就标 UI
- **正确**：
  - UI 测"飘字动画效果、位置"（归 UI）
  - HINT 测"飘字内容（+100 金币/-50 钻石）"（归 HINT B/C）
- **区分**：测"动画" → UI；测"数字/文案" → HINT

### 误判 3：道具图标显示 → 错标 UI

- **错误**：看到"图标"，就标 UI
- **正确**：
  - UI 测"图标渲染、显示效果"（归 UI）
  - CONFIG 测"图标路径字段、路径存在性"（归 CONFIG）
  - AUX 测"图标资源下载、缓存"（归 AUX）
- **区分**：测"渲染" → UI；测"路径格式" → CONFIG；测"下载" → AUX

### 误判 4：弹窗按钮点击 → 错标 UI

- **错误**：看到"弹窗"，就标 UI
- **正确**：
  - UI 测"弹窗布局、按钮样式"（归 UI）
  - HINT 测"弹窗文案内容、确认/取消逻辑"（归 HINT）
  - BIZ 测"按钮点击后的业务逻辑"（归 BIZ）
- **区分**：测"样式" → UI；测"内容/触发" → HINT；测"业务结果" → BIZ

### 误判 5：新手引导步骤 → 错标 UI

- **错误**：看到"引导"，就标 UI
- **正确**：
  - UI 测"引导遮罩颜色、高亮区域样式、引导步骤管理器 UI"（归 UI F）
  - HINT 测"引导文案内容、引导步骤顺序"（归 HINT H）
- **区分**：测"样式/管理器" → UI；测"内容/顺序" → HINT

### 误判 6：商城列表加载 → 错标 UI

- **错误**：看到"列表"，就标 UI
- **正确**：
  - UI 测"列表渲染、列表滑动、卡片样式"（归 UI）
  - BIZ 测"列表数据加载接口"（归 BIZ）
  - AUX 测"列表数据缓存"（归 AUX）
- **区分**：测"展示" → UI；测"数据" → BIZ；测"缓存" → AUX

### 误判 7：断网 UI 展示 → 错标 UI

- **错误**：看到"断网"，就标 UI
- **正确**：
  - UI 测"断网提示 UI 样式、位置"（归 UI H）
  - HINT 测"断网提示文案内容"（归 HINT）
  - SPECIAL 测"断网业务处理"（归 SPECIAL）
- **区分**：测"UI 样式" → UI；测"文案" → HINT；测"处理逻辑" → SPECIAL

## 4. 判定流程图

```
Story 涉及某个功能点
    │
    ├── 是"页面视觉/前端交互"？
    │     是 → 归 UI（按子类 A-H 选择）
    │           │
    │           ├── 控件渲染/状态/基础功能 → A（控件基础）
    │           ├── 纯前端本地交互 → B（纯前端交互）
    │           ├── 布局适配 → C（布局适配）
    │           ├── 静态展示 → D（静态展示）
    │           ├── 动效动画 → E（动效）
    │           ├── 引导提示样式 → F（引导）
    │           ├── 无障碍适配 → G（无障碍）
    │           └── 异常 UI 场景 → H（边界异常）
    │
    ├── 是"后端业务逻辑/接口"？
    │     是 → 归 BIZ
    │
    ├── 是"提示内容/触发条件"？
    │     是 → 归 HINT
    │
    ├── 是"配置表字段"？
    │     是 → 归 CONFIG
    │
    ├── 是"底层工具/框架"？
    │     是 → 归 AUX
    │
    ├── 是"跨服/第三方业务"？
    │     是 → 归 LINK
    │
    ├── 是"日志/埋点内容"？
    │     是 → 归 LOG
    │
    └── 是"特殊异常/对抗/合规"？
          是 → 归 SPECIAL
```
