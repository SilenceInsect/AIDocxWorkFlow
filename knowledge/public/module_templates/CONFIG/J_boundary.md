# J. CONFIG 模块边界区分（判定规则）

> **非测试类型**——本文件是 S5 生成测试点时的**判定参考**，
> 避免误标 CONFIG 标签（实际是 UI / BIZ / LINK / LOG / HINT）。
>
> **来源**：用户细化定义 §4「模块边界区分」

---

## 1. 判定原则

> CONFIG = **配置文件本身** + **导出/解析/热更/版本兼容** + **表与表之间数值/ID 约束** + **静态数值规则校验**
>
> **不归 CONFIG**：配置加载后的**运行时行为**、**业务逻辑**、**UI 展示**

### 判定三问

1. **是"配置本身"还是"运行时行为"？** 是运行时 → 不归 CONFIG
2. **是"静态数值规则"还是"业务逻辑计算"？** 是业务 → 不归 CONFIG
3. **是"导出/解析"还是"业务接口调用"？** 是业务 → 不归 CONFIG

## 2. 边界对照表

### vs BIZ（业务）

| 场景 | 归 CONFIG | 归 BIZ |
|------|-----------|--------|
| 道具表字段 | 字段类型、范围、ID 合法性 | 业务用道具发放、扣减逻辑 |
| 概率配置 | 概率总和 ≤ 100%、保底次数 | 抽卡逻辑、抽卡结果 |
| 商城定价 | 价格字段、单位、范围 | 购买逻辑、扣款、发货 |
| 货币配置 | 货币单位、转换比例 | 货币发放、扣减、流水 |
| 活动配置 | 活动起止时间、奖励内容 | 活动开启逻辑、奖励发放 |
| 任务配置 | 任务奖励道具 ID 存在 | 任务完成判定、奖励发放 |
| 礼包配置 | 礼包内容、限购次数 | 礼包购买、发放、限购逻辑 |
| 服务器倍率 | `server_exp_rate` 字段 | 玩家经验计算、实际发放 |

### vs UI

| 场景 | 归 CONFIG | 归 UI |
|------|-----------|-------|
| 道具图标 | 图标资源路径字段 | 图标显示、加载失败占位 |
| 道具名称 | 多语言文案字段 | 文案显示、字体、字号 |
| 弹窗内容 | 弹窗文案配置 | 弹窗打开/关闭、动画 |
| 商城布局 | 商城页布局配置 | 布局渲染、响应式 |
| 主题皮肤 | 皮肤资源 ID 配置 | 皮肤切换、样式应用 |
| 红点状态 | 红点状态字段 | 红点 UI 渲染 |

### vs LINK（关联）

| 场景 | 归 CONFIG | 归 LINK |
|------|-----------|---------|
| 多语言文案 | i18n key 字段、翻译表 | i18n 资源加载、跨语言切换 |
| 跨服务配置同步 | 配置同步协议字段 | 跨服务同步协议调用、冲突解决 |
| 第三方平台配置 | 第三方平台 ID 字段 | 调第三方平台（微信/支付宝） |
| CDN 资源 | CDN 路径字段 | CDN 资源下载、缓存 |

### vs LOG

| 场景 | 归 CONFIG | 归 LOG |
|------|-----------|--------|
| 配置变更 | 配置审计日志字段 | 运行时配置加载埋点 |
| 热更事件 | 热更版本号字段 | 热更事件埋点上报 |
| 错误配置 | 配置错误码字段 | 错误日志上报 |
| 配置访问 | 配置访问日志字段 | 访问埋点 |

### vs HINT

| 场景 | 归 CONFIG | 归 HINT |
|------|-----------|---------|
| 提示文案 | 提示文案字段 | 提示触发逻辑 |
| 红点状态 | 红点状态字段 | 红点显示/隐藏逻辑 |
| 飘字内容 | 飘字内容字段 | 飘字触发逻辑 |
| Toast 文案 | Toast 文案字段 | Toast 触发逻辑 |

### vs SPECIAL

| 场景 | 归 CONFIG | 归 SPECIAL |
|------|-----------|------------|
| 弱网配置 | 弱网超时、重试次数配置 | 弱网检测、降级策略 |
| 防作弊配置 | 防作弊阈值配置 | 防作弊检测、封号 |
| 反外挂配置 | 反外挂检测频率配置 | 反外挂检测逻辑 |

### vs UTIL

| 场景 | 归 CONFIG | 归 UTIL |
|------|-----------|--------|
| 资源路径 | 资源路径字段 | 资源下载、缓存、命中率 |
| 缓存策略 | 缓存 TTL 字段 | 缓存实现、淘汰策略 |
| 网络超时 | 网络超时字段 | 网络检测、自动重连 |

## 3. 常见误判案例

### 误判 1：抽卡保底 → 错标 CONFIG
- **错误**：看到"保底 90 次"，就标 CONFIG
- **正确**：
  - CONFIG 测"保底 90 次字段配置、生效"（归 CONFIG）
  - BIZ 测"玩家抽 90 次后必中 SSR 逻辑"（归 BIZ）
  - 同一 Story 应有 2 个 TP

### 误判 2：商城购买 → 错标 CONFIG
- **错误**：看到"商城价格 100 钻石"，就标 CONFIG
- **正确**：
  - CONFIG 测"商城定价字段、单位"（归 CONFIG）
  - BIZ 测"购买时扣 100 钻石、发货"（归 BIZ）
  - LINK 测"调第三方支付"（归 LINK）

### 误判 3：活动开启 → 错标 CONFIG
- **错误**：看到"活动起止时间"，就标 CONFIG
- **正确**：
  - CONFIG 测"活动起止时间字段、范围"（归 CONFIG）
  - I (SERVER_CONFIG) 测"定时任务按时开启/关闭"（归 CONFIG-I）
  - BIZ 测"活动开启后业务逻辑"（归 BIZ）

### 误判 4：热更生效 → 错标 CONFIG
- **错误**：看到"热更了某配置"，就标 CONFIG
- **正确**：
  - CONFIG-D 测"热更流程、生效"（归 CONFIG）
  - BIZ 测"新配置生效后的业务行为"（归 BIZ）
  - LOG 测"热更事件埋点"（归 LOG）

### 误判 5：防沉迷 → 错标 CONFIG
- **错误**：看到"防沉迷 90 分钟"，就标 CONFIG
- **正确**：
  - CONFIG-I 测"防沉迷阈值字段"（归 CONFIG）
  - SPECIAL 测"防沉迷检测、踢出"（归 SPECIAL）

## 4. 判定流程图

```
Story 涉及某个功能点
    │
    ├── 是"配置本身"（字段/枚举/资源/ID/时间）？
    │     是 → 归 CONFIG（按子类 A/B/C/I 选择）
    │
    ├── 是"配置导出/解析/热更"？
    │     是 → 归 CONFIG（按子类 D/E/F/H 选择）
    │
    ├── 是"配置数值公式/概率/上限"？
    │     是 → 归 CONFIG（按子类 G 选择）
    │
    ├── 是"服务端业务配置"（开服/防沉迷/GM）？
    │     是 → 归 CONFIG（按子类 I 选择）
    │
    ├── 是"配置版本兼容/灰度"？
    │     是 → 归 CONFIG（按子类 F 选择）
    │
    ├── 是"业务逻辑"（发放/扣减/计算/触发）？
    │     是 → 归 BIZ
    │
    ├── 是"配置加载后 UI 展示"？
    │     是 → 归 UI
    │
    ├── 是"跨服务/外部系统"？
    │     是 → 归 LINK
    │
    ├── 是"埋点/审计/日志"？
    │     是 → 归 LOG
    │
    └── 是"提示内容/状态"？
          是 → 归 HINT
```
