# O. CONFIG 模块边界区分（判定规则）

> **非测试类型**——本文件是 S5 生成测试点时的**判定参考**，
> 避免误标 CONFIG 标签（实际是 UI / BIZ / AUX / LINK / LOG / SPECIAL / HINT）。

---

## 1. 判定原则

> **CONFIG = 配置表字段合法性 + 同表/跨表数值约束 + 热更 + 版本兼容 + 导出发布 + 服务端配置**
>
> **不归 CONFIG**：UI 渲染、业务逻辑、底层工具、跨服同步、日志内容

### 判定三问

1. **是"配置文件本身的字段/数值/格式"还是"业务逻辑"？** 是业务逻辑 → 不归 CONFIG
2. **是"配置层"还是"渲染/展示层"？** 是渲染 → 不归 CONFIG
3. **是"配置内容"还是"配置变更后的业务效果"？** 是业务效果 → 不归 CONFIG

## 2. 边界对照表

### vs UI

| 场景 | 归 CONFIG | 归 UI |
|------|-----------|-------|
| 道具图标 | 图标路径字段格式校验、路径存在性 | 图标加载、渲染、显示效果 |
| 皮肤配置 | 皮肤表字段、皮肤 ID 合法性 | 皮肤切换动效、皮肤 UI 展示 |
| 弹窗文案 | 文案 key 字段、国际化 key 映射 | 弹窗样式、布局、按钮样式 |
| 颜色配置 | 颜色值字段（#RRGGBB）| 颜色在 UI 上的应用效果 |
| 字体配置 | 字体名称/路径字段 | 字体渲染效果 |

### vs BIZ

| 场景 | 归 CONFIG | 归 BIZ |
|------|-----------|-------|
| 概率 | 概率字段范围 0-1、概率总和 ≤ 100% | 抽卡业务逻辑、保底机制 |
| 数值公式 | 公式配置、公式字段格式 | 公式计算后的业务结果 |
| 限购 | 限购数量字段、限购类型枚举 | 限购业务逻辑、购买判断 |
| 奖励 | 奖励表字段、奖励 ID 合法性 | 奖励发放业务逻辑 |
| 活动 | 活动起止时间字段 | 活动开启/结束业务逻辑 |
| 抽卡 | 抽卡概率配置、保底配置 | 抽卡流程、保底计数 |
| 数值上限 | 上限数值字段 | 超上限后的业务处理 |

### vs AUX

| 场景 | 归 CONFIG | 归 AUX |
|------|-----------|-------|
| 热更新 | 配置文件热更（道具表/活动表）| 客户端版本热更框架、资源热更 |
| 解析加载 | 配置表解析逻辑 | 解析 SDK、JSON 工具 |
| 资源路径 | 资源路径字段格式 | 资源下载、CDN 路径 |
| 导出工具 | Excel → JSON 导出 | 导出工具的 SDK/框架 |

### vs LINK

| 场景 | 归 CONFIG | 归 LINK |
|------|-----------|-------|
| 跨服 ID | 跨服 ID 格式、字段合法性 | 跨服同步时序、数据一致性 |
| 第三方 ID | 第三方渠道 ID 字段 | 第三方登录/支付对接 |
| 灰度配置 | 灰度服开关字段 | 灰度流量调度 |

### vs LOG

| 场景 | 归 CONFIG | 归 LOG |
|------|-----------|-------|
| 配置变更 | 配置变更日志记录（audit log）| 配置变更的内容本身 |
| 导出日志 | 导出工具操作日志 | 配置表内容 |
| 性能日志 | 配置加载耗时上报 | 配置字段内容 |

### vs SPECIAL

| 场景 | 归 CONFIG | 归 SPECIAL |
|------|-----------|-----------|
| 版本兼容 | 新旧版本配置字段兼容 | 版本不兼容的业务异常处理 |
| 防沉迷 | 防沉迷时间配置字段 | 防沉迷校验业务逻辑 |

## 3. 常见误判案例

### 误判 1：道具图标显示错误 → 错标 CONFIG

- **错误**：看到"图标"，就标 CONFIG
- **正确**：
  - CONFIG 测"图标路径字段格式、路径存在性"（归 A）
  - UI 测"图标渲染、显示效果"（归 UI）
  - AUX 测"图标资源下载、缓存"（归 AUX）

### 误判 2：抽卡概率超 100% → 错标 BIZ

- **错误**：看到"概率"，就标 BIZ
- **正确**：
  - CONFIG 测"概率字段范围 0-1"（归 A）
  - CONFIG 测"所有概率总和 ≤ 100%"（归 G）
  - BIZ 测"抽卡业务逻辑"（归 BIZ）

### 误判 3：热更新后道具显示错误 → 错标 CONFIG

- **错误**：看到"热更"，就标 CONFIG
- **正确**：
  - CONFIG 测"配置热更流程"（归 D）
  - BIZ 测"热更后业务生效逻辑"（归 BIZ）
  - UI 测"热更后 UI 刷新"（归 UI）

### 误判 4：皮肤切换效果错误 → 错标 CONFIG

- **错误**：看到"皮肤"，就标 CONFIG
- **正确**：
  - CONFIG 测"皮肤表字段、皮肤 ID 合法性"（归 A）
  - UI 测"皮肤切换动效、UI 展示"（归 UI）

### 误判 5：弹窗文案显示错误 → 错标 CONFIG

- **错误**：看到"弹窗文案"，就标 CONFIG
- **正确**：
  - CONFIG 测"文案 key 字段、国际化 key 映射"（归 A）
  - UI 测"弹窗样式、布局"（归 UI）
  - HINT 测"弹窗文案内容"（归 HINT）

## 4. 判定流程图

```
Story 涉及某个功能点
    │
    ├── 是"配置表字段/数值/格式"？
    │     是 → 归 CONFIG（按子类 A-I 选择）
    │           │
    │           ├── 字段类型/取值/枚举 → A（字段合法性）
    │           ├── 同表多字段数值一致 → B（同表一致性）
    │           ├── 跨表 ID 引用存在 → C（跨表依赖）
    │           ├── 配置热更 → D（热更新）
    │           ├── 配置解析加载性能 → E（解析与加载）
    │           ├── 版本间配置兼容 → F（版本兼容）
    │           ├── 概率/公式/上限 → G（数值逻辑）
    │           ├── 导出打包发布 → H（导出发布）
    │           └── 服务端常量/定时 → I（服务端专属）
    │
    ├── 是"UI 渲染/展示"？
    │     是 → 归 UI
    │
    ├── 是"业务逻辑"？
    │     是 → 归 BIZ
    │
    ├── 是"底层工具/框架"？
    │     是 → 归 AUX
    │
    ├── 是"跨服/第三方业务"？
    │     是 → 归 LINK
    │
    ├── 是"日志/埋点内容"？
    │     是 → 归 LOG
    │
    ├── 是"特殊异常/对抗/合规"？
    │     是 → 归 SPECIAL
    │
    └── 是"提示内容/文案"？
          是 → 归 HINT
```
