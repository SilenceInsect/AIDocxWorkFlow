# AMRD · 按钮 4 态 Override（移动端基线）

> **本文件是 AMRD 项目对 `module_templates/UI/A_control_basic.md` 的 override**。
>
> **优先级**：本项目内本子类 override **优先于**通用库。
>
> **来源**：用户原话（2026-07-22 凌晨）—— *"AMRD 按钮没有 8 态，8 态是网页的，手游一般只有常态，点击态，置灰，高亮（有奖励可领取等）"*

---

## 1. 状态数：8 → 4

| 维度 | 通用库（网页基线）| **AMRD（移动端基线）** |
|------|---------------|----------------------|
| 状态数 | 8 | **4** |
| 状态列表 | IDLE / HOVERED / FOCUSED / PRESSED / SELECTED / DISABLED / LOADING / DRAGGING | **IDLE / PRESSED / DISABLED / ACTIONABLE** |
| 设计哲学 | 覆盖所有交互设备 | **只覆盖移动端真实交互** |

---

## 2. 4 态定义（专业术语）

### 2.1 IDLE（默认态）

| 项 | 内容 |
|----|------|
| **定义** | 控件存在、可交互、未被操作的常态 |
| **触发** | 页面加载完成 / 操作完成后 |
| **视觉** | 默认配色 / 默认文案 / 默认图标 |
| **用户操作** | 可点击 / 可长按 / 可拖拽 |
| **验证要点** | 加载完成后立即进入 IDLE / 操作完成后回归 IDLE / 不同皮肤下 IDLE 都正确 |

### 2.2 PRESSED（按下态 / 高亮瞬态）

| 项 | 内容 |
|----|------|
| **定义** | 手指按下瞬间的视觉反馈（瞬态）|
| **触发** | `TouchDown` 事件 |
| **视觉** | 颜色加深 / 透明度降低 / 缩放 0.95-0.98 / 短促涟漪动画 |
| **用户操作** | 按下后立即松开 = 单击；按下后停留 = 长按 |
| **验证要点** | 按下视觉反馈 < 100ms / 松开后立即回 IDLE / 长按时 PRESSED 持续 / 多点触控不冲突 |

### 2.3 DISABLED（禁用态 / 不可交互态）

| 项 | 内容 |
|----|------|
| **定义** | 控件存在但不可交互（"置灰"）|
| **触发** | 业务条件不满足（CD 未到 / 余额不足 / 等级不够 / 网络离线）|
| **视觉** | 灰色 / 透明度 0.4-0.6 / 无阴影 / 无涟漪 |
| **用户操作** | 点击无响应（**不被穿透**）/ 长按可显示原因（可选）|
| **验证要点** | 灰度 / 透明度符合设计稿 / 点击不响应（不被穿透到下层） / 业务条件恢复后**自动**回 IDLE |

### 2.4 ACTIONABLE（可行动态 / 待办态）

| 项 | 内容 |
|----|------|
| **定义** | 控件有效且需引起玩家注意（"高亮" / "有奖励可领取" / "闪烁" / "跳动"）|
| **触发** | 有奖励待领取 / 有任务待完成 / 有消息未读 / 有活动可参加 |
| **视觉** | 脉冲动画 / 呼吸光效 / 红点 / 角标 / 闪烁 |
| **用户操作** | 点击进入对应流程（领取 / 任务 / 活动）|
| **验证要点** | 红点 / 角标数字准确 / 脉冲节奏不扰民 / **进入流程后红点消失**（自动消）|

## 3. 显式排除的通用状态（含理由）

| 排除状态 | 排除理由（移动端特有）|
|---------|---------------------|
| **HOVERED**（悬停态）| 手指没有"悬停"——touch-based UI 无此概念。**鼠标外接场景罕见**。 |
| **FOCUSED**（焦点态）| Tab 焦点在移动端语义弱（外接键盘场景外罕见）。**焦点环若强制显示会误导用户**。 |
| **SELECTED**（选中态）| 单选 / 多选由独立组件管（`RadioGroup` / `CheckBox`）。**按钮本身不放 SELECTED**。 |
| **LOADING**（加载态）| **不是按钮状态**——是内容状态。按钮不放 loading，用 **skeleton / spinner / Toast**。 |
| **DRAGGING**（拖拽态）| AMRD 按钮默认不可拖拽（**拖拽由独立 DragHandle 组件承载**）。 |

> 详细决策记录见 [`../_decisions.md`](../_decisions.md) #1。

## 4. AMRD 按钮种子 TP（4 态 × 4 类型 = 16 条核心）

> **每条 TP 显式标注规则来源**：`source: project_local/AMRD/ui/A_button_mobile_4state.md`

### 4.1 IDLE（默认态）· 4 条

| TP ID | 类型 | 描述 | 验证要点 |
|------|------|------|---------|
| AMRD-UI-A-001-TP-001 | POSITIVE | 页面加载完成后按钮进入 IDLE | 视觉符合设计稿 / 可点击 |
| AMRD-UI-A-001-TP-002 | BOUNDARY | 皮肤切换后按钮 IDLE 正确 | 深色皮肤 / 浅色皮肤 / 节日皮肤 |
| AMRD-UI-A-001-TP-003 | NEGATIVE | IDLE 状态误显示 PRESSED（视觉残留）| 操作完成后 200ms 内回 IDLE |
| AMRD-UI-A-001-TP-004 | EXCEPTION | 页面加载失败时按钮 IDLE 行为 | 灰显 + Toast 错误提示 |

### 4.2 PRESSED（按下态）· 4 条

| TP ID | 类型 | 描述 | 验证要点 |
|------|------|------|---------|
| AMRD-UI-A-002-TP-001 | POSITIVE | 按下瞬间视觉反馈 | < 100ms / 颜色加深 / 涟漪 |
| AMRD-UI-A-002-TP-002 | BOUNDARY | 长按时 PRESSED 持续 | 长按 2s 期间保持 PRESSED |
| AMRD-UI-A-002-TP-003 | NEGATIVE | 多点触控冲突 | 两指同时按下不冲突 |
| AMRD-UI-A-002-TP-004 | EXCEPTION | 手指滑出按钮区域时 PRESSED 取消 | 滑出后立即回 IDLE（不触发点击）|

### 4.3 DISABLED（禁用态）· 4 条

| TP ID | 类型 | 描述 | 验证要点 |
|------|------|------|---------|
| AMRD-UI-A-003-TP-001 | POSITIVE | 业务条件不满足时显示 DISABLED | 灰显 / 透明度符合设计稿 |
| AMRD-UI-A-003-TP-002 | BOUNDARY | DISABLED 状态下点击不响应（不被穿透）| 点击无响应 / 不触发下层 |
| AMRD-UI-A-003-TP-003 | NEGATIVE | DISABLED 误判（业务条件已恢复但仍 DISABLED）| 业务恢复后**自动**回 IDLE |
| AMRD-UI-A-003-TP-004 | EXCEPTION | 离线状态下所有按钮都 DISABLED | 断网时统一置灰 + 顶部 Toast |

### 4.4 ACTIONABLE（可行动态）· 4 条

| TP ID | 类型 | 描述 | 验证要点 |
|------|------|------|---------|
| AMRD-UI-A-004-TP-001 | POSITIVE | 有奖励待领取时显示 ACTIONABLE | 红点 / 脉冲动画 / 角标数字 |
| AMRD-UI-A-004-TP-002 | BOUNDARY | 角标数字 99 / 99+ 边界 | ≥100 显示 99+ |
| AMRD-UI-A-004-TP-003 | NEGATIVE | ACTIONABLE 误判（已领完但仍 ACTIONABLE）| 进入流程后红点**立即**消失 |
| AMRD-UI-A-004-TP-004 | EXCEPTION | 多奖励同时 ACTIONABLE 不冲突 | 红点数字累加正确 / 优先级正确 |

## 5. AMRD 通用模板的额外建议（**不是 override，是补充**）

| 建议 | 内容 |
|------|------|
| **按钮文案长度** | AMRD 按钮文案 ≤ 6 个汉字（防 i18n 溢出）|
| **按钮尺寸** | 最小可点击区域 44×44pt（iOS HIG）/ 48×48dp（Material Mobile）|
| **按钮间距** | 按钮间 ≥ 8pt（防误触）|
| **按钮动效** | PRESSED 涟漪 ≤ 200ms（手游快节奏）|
| **按钮音效** | PRESSED 时播放短促点击音（可关）|
